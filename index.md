# SurfWave

**High‑performance, energy‑efficient wireless communication enabled by intelligent radio‑frequency computation.**

SurfWave is developing a new class of antennas powered by an intelligent computation layer that performs part of the signal processing directly in the radio‑frequency domain. This architecture enables higher throughput, better coverage efficiency, and significantly lower energy consumption compared to traditional 5G systems.

SurfWave’s mission is to build sustainable, high‑performance wireless communication infrastructure that expands access to reliable connectivity.
